# :bell:SpringBoot와 JSP을 통한 게시판 구현
<br>
<br>


## :dvd:개발 환경
* Java 11
* MyBatis
* Spring Boot 2.7.8
* Maven
* OracleDB

<br>

## :wrench:개발 기술 스택
* JSP
* Spring Boot
* CSS
* JavaScript
* Java

<br>

## :computer:게시판 구현 기능
* CRUD 기능 [ 생성, 조회, 수정, 삭제 ]
* 조회수 카운트 기능
* 게시글 목록 Paging 처리
